Projects

Friendle

In a team of 4, I worked on the backend for a matching/hangout suggestion app to help those with social anxiety. We deployed the backend on GCP Cloud Functions (server-less) and integrated it with the Firestore database. Our project also received 1st place for the Telus Mental Health Hack and 2nd place overall at UoftHacks VIII.

Check us out on Devpost!

Recordr.ai: Face-tracking Video Recorder

In a team of 4, I helped develop an automated video recording app. We used FaceNet and PoseNet, implemented in Tensorflow and hosted on an EC2 instance, to recognize the speakers' faces and detect gesture-based commands. To communicate with the neural nets, we used AWS IoT to receive real-time commands from the EC2 and relay them to an Arduino motor with a smartphone mount.
You can see a demo of the project here: Google Drive

Search Engine & Website Crawler (Python)

I created a search engine with the PageRank algorithm. I used Bottle and Google OAuth to handle sessions and allow users to sign-in using their Google credentials. The backend crawled webpages, made queries using SQLite and was hosted on an AWS EC2 instance.

Crunch: A reaction-based video game (Verilog)

In a team of two, my partner and I created a game where users control an alligator to avoid obstacles. The game used multiple FSMs (Finite State Machine) for the graphics and varying levels.
